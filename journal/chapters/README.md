# Chapter Narratives

Two documents per chapter:

**`chapter-N-live.md`** — Live trajectory, updated after every session. Contains trajectory status (ASCENDING / STABLE / DRIFTING / DESCENDING / CRISIS), signal log, predicted dangers.

**`chapter-N-postmortem.md`** — Written when chapter closes. Timeline, compliance curve, pattern activations, root cause, lessons, carry-forward items.

Create `chapter-1-live.md` during your first trading session. The AI will maintain it from there.
