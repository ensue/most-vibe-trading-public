# Position journal — optional plans & history

Plans are **optional**. After `python exchange/sync.py`, `exchange/data/` holds the factual record of open positions.

When you want a written plan, ask MOST to create `YYYY-MM-DD-<symbol>-<side>-<slug>.md`.

Suggested fields when logging: symbol, side, entry, size, SL, TP, risk USD, thesis.

If you ask, MOST can compare a saved note to the latest snapshot — **neutral diff**, not approval/rejection.
